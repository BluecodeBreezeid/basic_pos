## ⚙️ Setup

<!-- Tekan Ctrl + Shift + V untuk melihat -->

1. Extract project ke folder `htdocs`
2. Import database dari file `database.sql`
3. Pastikan konfigurasi database di file `.env` sudah sesuai
4. Akses aplikasi melalui:
   http://localhost/basic_pos/public/login
